### Metrics framework drafted (for the admin dashboard)

Categories the acquisition-readiness dashboard will show:

| Category | Why it matters to a buyer |
|---|---|
| **Audience** (MAU, DAU, DAU/MAU, growth rate) | Slope > absolute at this scale |
| **Dataset** (total ratings, unique fights with ≥10 ratings, oldest rating date, promotion coverage) | The 10-year moat — quantified |
| **Engagement** (ratings per MAU, notification opt-in %, avg followed fighters) | Proves users actually use it |
| **Live events** (peak concurrency per card, ratings/min during main event) | The fight-night demo metric — needs PostHog |
| **Operational** (crash-free rate, backend uptime, cost per MAU) | Bus-factor + business viability |
