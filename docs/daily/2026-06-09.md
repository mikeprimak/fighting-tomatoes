**Key OpenShot gotchas learned (logged in build-log):**
- SFX must be **File → Import Files**'d into the project before they appear in Project Files — staged-on-disk ≠ imported.
- Keyframes are anchored **relative to the clip** — trimming the left edge drags them along, so don't build "holds" by trimming. Place keyframes in the back half and let OpenShot hold the first keyframe's value backward.
- **OpenShot scales from the CLIP CENTER.** `Origin X` only affects *rotation*, not scale — so the spec's "Origin X=0" left-anchor trick does NOT work (bar grows bidirectionally from center). The **Crop effect also failed** (revealed an empty container). **Solution:** bake the anchor into the PNG — gold lives in the RIGHT half of a 1080-wide transparent canvas so its left edge sits at image-center = the scale pivot → pure Scale X 0→0.98 pins the left edge, zero drift. Assets: `rating-bar-{gold,rail}-leftanchor-1080x20.png`.
- Step-by-step UI instructions repeatedly missed; **describing the target look/sound and letting Mike build it** worked better for the motion-feel elements.
