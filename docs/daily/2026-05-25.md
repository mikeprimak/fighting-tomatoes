   `/api/events` pattern. Backend typecheck (production config) clean.
   ✅ **VERIFIED live at session end:** anonymous `/api/fighters/<id>` now returns
   fighter JSON, and the web fighter page renders the fighter's name as `<title>`
   + a real og:image (was `Fighter — Good Fights` with none).

   One-line trim fix if we want it; not done.
2. **`fix(web)` `ba19724` — double `— Good Fights` title suffix.** `layout.tsx`
