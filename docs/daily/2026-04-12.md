## Changes Made

### Backend: UFC-first sorting on Past Events page

**File**: `packages/backend/src/routes/index.ts`

Previously, past events used a simple `date DESC` Prisma orderBy, so events on the same day appeared in arbitrary order. Upcoming events already had an in-memory sort that puts UFC events above other orgs on the same calendar day.

Applied the same pattern to past events: fetch a lightweight list of all matching events, sort in-memory by day descending then UFC-first within each day, paginate, then load full data for that page. This ensures UFC 327 shows above RIZIN FF, PFL Chicago, Oktagon 86, etc. when they fall on the same date.
