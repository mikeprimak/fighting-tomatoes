## Android Play Console — 2 deprecation warnings flagged (deferred, address later)

Play Console flagged **2 "recommended actions"** on the 2.1.0 Android release (P2).
Neither blocks release; both are forward-compat warnings. Logging for a later pass.

### 1. Deprecated edge-to-edge / window APIs (Android 15)

Play reports these deprecated APIs in use:
- `android.view.Window.setStatusBarColor` / `setNavigationBarColor` / `getStatusBarColor` / `getNavigationBarColor`
- `LAYOUT_IN_DISPLAY_CUTOUT_MODE_SHORT_EDGES`, `LAYOUT_IN_DISPLAY_CUTOUT_MODE_DEFAULT`

Origins are all **third-party/framework code**, not our app source:
- `com.facebook.react.views.view.WindowUtilKt.enableEdgeToEdge` / `statusBarHide` / `statusBarShow` (React Native core)
- `com.google.android.material.bottomsheet.BottomSheetDialog.onCreate`, `EdgeToEdgeUtils.applyEdgeToEdge`, `sidesheet.SheetDialog.onCreate` (Material)
- `com.swmansion.rnscreens.ScreenWindowTraits.setColor` / `setNavigationBarColor` (react-native-screens)
- `expo.modules.devlauncher...DevLauncherExpoActivityConfigurator.setColor` (dev-launcher, dev-only)
- `expo.modules.imagepicker.ExpoCropImageUtils.applyWindowTheming` (expo-image-picker)

**Assessment:** these resolve as RN/Expo/Material upgrade away from the deprecated
window-color APIs. Not actionable in our code today — fix is to bump Expo SDK / RN /
react-native-screens once their releases drop the deprecated calls. Revisit on next
SDK upgrade.

### 2. Resizability / orientation restrictions (Android 16)

From Android 16, large-screen devices (foldables, tablets) will **ignore** orientation
restrictions. Play detected portrait locks on:
- `com.google.mlkit.vision.codescanner.internal.GmsBarcodeScanningDelegateActivity` (PORTRAIT) — third-party (ML Kit)
- `com.fightcrewapp.mobile.MainActivity` (PORTRAIT) — **ours**, from `app.json` `"orientation": "portrait"`

**Assessment:** the app is designed portrait-only (`supportsTablet: false` on iOS too),
so this is intentional, but on Android 16 tablets/foldables it'll render in landscape
regardless and may show layout issues. If we ever care about large-screen Android,
test landscape layouts and relax the lock. Low priority — combat-sports phone app,
no tablet story today.
