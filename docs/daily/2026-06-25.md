# 2026-06-25

## Home "This Week" event cards: hype-fight count + shorter blurbs

Two related changes to the mobile Home tab's "Events by day" cards (Today /
Tomorrow / Saturday …), both in `packages/mobile/app/(tabs)/home.tsx`.

### 1. "N hype fights" + top 2 hyped matchups on each event card

Under the AI "why care" blurb, every by-day event card now shows a
`N hype fight(s)` line plus up to the top 2 qualifying matchups (with their
heatmap-colored avg hype score).

- A fight qualifies as "hyped" when it has **>= 3 hype predictions AND a >= 7.5
  average** (`HYPE_MIN_COUNT` / `HYPE_MIN_AVG` + `getHypeFights()` helper in
  home.tsx). Sorted by avg hype desc; count is the full qualifying set, list is
  capped at 2.
- This needs per-fight hype aggregates, so the home upcoming-events query was
  flipped from `includeFights: false` → `true` (`averageHype` / `hypeCount` come
  from the backend `/api/events?includeFights=true` aggregation over
  `FightPrediction.predictedRating`). Same backend work the Events screen already
  does. Slight extra load on the Home query — fine at current scale; revisit if
  the events query ever becomes a fight-night hot spot.
- `EventRow` gained an optional `hypeFights?: any[]` prop. Only the by-day cards
  pass it; the "Event Last Night" card deliberately does not (still
  `includeFights: false`, no hype line).
- New styles: `eventRowHype*`.

### 2. Rewrote existing event blurbs to single <= 10-word sentences

The backend AI-enrichment prompt was already updated to emit short blurbs going
forward; this was a one-off cleanup of the long ones already in the DB. Only
confidence-gated (>= 0.5) `aiEventSummary` values render, so only those were
rewritten (prod DB text UPDATE, 5 events that are currently showing):

| Event | New blurb |
|---|---|
| ONE Friday Fights 160 | Two undefeated strikers clash for the vacant flyweight title. |
| UFC FN Fiziev vs. Torres | Fiziev defends home turf against dangerous finisher Torres. |
| PFL San Diego 2026 | McKee returns from upset to face undefeated Isbulaev. |
| BKFC Liberty Brawl | Former champion Trout headlines a lightweight title fight. |
| Pflmena10 | Rising prospect Alsaif debuts atop a stacked Saudi card. |

Also added `numberOfLines={2}` to `eventRowDesc` as a display guard.
