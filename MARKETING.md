## Event-Based Workflow

### When a Major Event is Announced

**Immediately**:
- [ ] Add event date to calendar
- [ ] Set reminders for content days (see fight week calendar)

**1 Week Before Event**:
- [ ] Check app for current hype scores
- [ ] Screenshot top 3-5 hyped fights
- [ ] Create "Most Hyped" carousel
- [ ] Schedule for Monday of fight week

**3 Days Before**:
- [ ] Update hype scores (they may have changed)
- [ ] Create individual fight hype cards for main event + co-main
- [ ] Post "Card Preview" content

**Day Before**:
- [ ] Final hype score update
- [ ] Post "Final Predictions" content
- [ ] Prepare post-fight templates (leave ratings blank)

**Event Day**:
- [ ] Watch fights with phone ready
- [ ] Post real-time ratings after each main card fight
- [ ] Instagram Stories: Polls asking "Rate this fight"

**Day After**:
- [ ] Collect final ratings from app
- [ ] Create "How the Card Scored" summary
- [ ] Post by noon
- [ ] Engage with comments all day

## Growth Tactics

### Ongoing (Do These Regularly)

1. **Engage with MMA community**:
   - Comment on UFC/fighter posts
   - Reply to people discussing fights
   - Don't spam - be genuinely helpful ("Our app shows the crowd rated that 8.7!")

2. **Partner with podcasters/YouTubers**:
   - DM small-medium MMA podcasts
   - Offer: "We'll give you exclusive hype data for your preview shows"
   - They mention app, you provide data = win-win

3. **Reddit (careful)**:
   - Don't spam r/MMA with app links
   - DO share interesting data as infographics
   - Comment naturally, mention app only when relevant

4. **Respond to current events**:
   - Controversial decision? Post the crowd rating
   - Viral KO? Post the rating and ask for more
   - Fighter callout? Show their recent ratings

### One-Time Growth Campaigns

**Campaign 1: Prediction Challenge (Month 2)**
- Partner with MMA influencer
- They predict card, compare to app's crowd prediction
- "Can [Influencer] beat the FightCrewApp crowd?"
- Drives downloads + content

**Campaign 2: FOTY Voting Push (End of Year)**
- "Only on FightCrewApp can you vote for Fight of the Year"
- Weekly updates on FOTY race
- Big reveal post on December 31

**Campaign 3: User Testimonials (Month 3+)**
- Ask active users for video testimonials
- "Why I use FightCrewApp to rate fights"
- Authentic content performs well

## Metrics & Tracking

### Weekly Check (Every Sunday)

| Metric | Where to Find | Target |
|--------|---------------|--------|
| Instagram followers | Profile | +50/week initially |
| Instagram engagement rate | Post insights | >3% |
| X followers | Profile | +30/week initially |
| X impressions | Analytics | Growing week over week |
| App downloads | App Store Connect / Google Play Console | Track correlation with posts |

### Monthly Review

1. **Best performing post**: What made it work? Do more like it.
2. **Worst performing post**: Why? Stop doing that.
3. **Follower growth**: On track? Adjust if not.
4. **Download correlation**: Did big posts = more downloads?

### Simple Tracking Spreadsheet

Create a Google Sheet with these columns:
- Date
- Platform
- Content Type
- Topic/Fight
- Likes
- Comments
- Shares
- Notes

Fill in weekly. Look for patterns monthly.
