# Fighting Tomatoes

A React Native + Node.js application for rating combat sports fights.

## Tech Stack
- React Native (Expo)
- Node.js + Express + TypeScript
- PostgreSQL + Prisma ORM
- JWT Authentication

## Development
```bash
# Install dependencies
pnpm install

# Start backend
cd packages/backend && pnpm dev

# Start mobile app
cd packages/mobile && pnpm start
